#Демо экзамен
Приложение создано для просмотра товаров и их данных.
В приложении существует одна страница, в ней находиться весь функционал.
Для поиска нужного товара возможно одновременно использовать поисковик, который работает по наименованиям товаров, список товаров можно отсортировать и отфильтровать. 
На странице присутсвует нумерация страниц для комфортного отображения списка товаров.
